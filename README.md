# 🧾 生活辦事英文

一個**離線可用、免安裝、零追蹤**的生活辦事會話 PWA，收錄銀行、郵局、看病、租屋、辦門號等「出門辦事」情境的實用英文短句。比照「旅遊英文」的會話引擎製作。

> 純單一 HTML 檔、資料全存本機、不上傳雲端、免註冊、無廣告。

> **狀態**：✅ 已上線　｜　🟠 橘色主題（theme `#ea580c`）　｜　**13 情境**（與「社交英文」合計 208 句）　｜　📱 響應式
> 上線網址：<https://fullmodel-star.github.io/errand-english/>　｜　已收錄於[英語學習入口 hub](../14_英語學習入口/)

---

## ✨ 功能
- **13 個辦事情境**：每句含英文、中文、諧音、朗讀（比照旅遊引擎）
- 情境瀏覽、收藏、口說練習與回饋

## 🚀 使用
瀏覽器開 `index.html` 即可；安裝成 App 需用上線 https 網址（Android Chrome →「安裝應用程式」，iPhone Safari →「加入主畫面」）。

## 📁 檔案結構
| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（單檔即整個 App） |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 |
| `icon.svg` / `icon-*.png` / `apple-touch-icon.png` | 圖示 |

## 🔒 隱私
完全離線、不連外部伺服器；資料只存裝置 `localStorage`；無帳號、無追蹤、無廣告。
